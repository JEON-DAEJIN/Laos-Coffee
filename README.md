# Lao Aroma - 라오스 유기농 커피 여행 스토어

**[🛍️ 쇼핑몰 바로가기](https://laos-coffee.vercel.app)**

라오스에서 생산한 유기농 커피(원두·분쇄커피·커피믹스)를 판매하여, 라오스·베트남·캄보디아를 여행했던 사람들이 여행의 추억과 커피의 향을 다시 느낄 수 있도록 하는 온라인 쇼핑몰입니다.

> "커피를 판매하는 것이 아니라 여행의 기억을 다시 선물한다."

자세한 기획 배경과 요구사항은 [`docs/PRD v1.0 - 라오스 유기농 커피 여행 쇼핑몰.docx`](./docs/PRD%20v1.0%20-%20라오스%20유기농%20커피%20여행%20쇼핑몰.docx) 문서를 참고하세요. 이 README는 PRD를 기반으로 한 개발 진행용 요약본이며, 진행 상황에 맞춰 계속 업데이트합니다.

## 프로젝트 개요

- **타깃 사용자**: 라오스·베트남·캄보디아 여행 경험이 있는 20~40대, 해외여행/현지커피/유기농 식품을 선호하는 사람
- **핵심 가치**: 여행의 추억을 다시 경험하게 하는 커피 쇼핑몰
- **개발 범위(현재)**: 대부분 프런트엔드 중심으로 진행하되, 리뷰·로그인은 Supabase(Postgres + Auth)로 실제 저장/인증됩니다. 결제 연동은 이후 단계에서 진행

## 실행 방법

```bash
npm install
npm run dev
```

`http://localhost:3000`에서 확인합니다.

## 배포

[Vercel](https://vercel.com)에 배포되어 있습니다 (`vercel deploy --prod`). 커밋을 push한 뒤 재배포하려면 위 명령을 다시 실행합니다.

## 백엔드 (Supabase)

과제("내 서비스에 백엔드 붙이기") 갈래 B로, 리뷰 작성/조회와 로그인에 실제 백엔드(Supabase)를 연결했습니다.

| 항목 | 내용 |
|---|---|
| 백엔드가 꼭 필요한 기능 | 리뷰 작성/조회 — 새로고침·다른 기기에서도 남아야 함 |
| 화면만으로 되는 기능 | 상품 목록/상세(목업 데이터), 장바구니, 주문·결제(로컬 시뮬레이션) |
| 엔드포인트 | 리뷰 목록 조회(Read), 리뷰 작성(Create) — Supabase 자동 생성 REST API |
| 비밀값 | `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY` → `.env.local` / Vercel 환경변수로 분리 (anon/publishable 키는 공개되어도 되는 키) |
| 호스팅 | Supabase (BaaS) — 저장·조회·로그인이 전부라 별도 서버 불필요 |

**스키마**: [`docs/dev-journey/supabase-reviews-schema.sql`](./docs/dev-journey/supabase-reviews-schema.sql)

**보안 체크**
- [x] 비밀값은 코드에 직접 안 적고 환경변수로 분리
- [x] 리뷰 작성은 백엔드(RLS)에서 검증 — `curl`로 인증 없이 insert 시도 시 401 확인
- [x] RLS 활성화 + 정책 적용 (조회는 공개, 작성은 `auth.uid() = user_id`인 경우만)
- [x] 로그인 필요한 동작(리뷰 작성)에 인가 체크 포함
- [x] 무료 티어 한도 확인 (Supabase Free: DB 500MB, MAU 5만명, 1주 미사용 시 프로젝트 일시정지)

## 기술 스택

- **Framework**: Next.js (React 기반)
  - SEO 노출에 유리 (라오스 커피, 유기농 커피, 여행 선물 등 키워드 검색 대응)
- **Backend**: Supabase (Postgres + Auth) — 리뷰·로그인만 연결, 나머지는 목업/로컬 상태
  - AI 코드 생성 안정성이 높고 한국어 학습 자료가 풍부
  - 로그인/회원가입/장바구니/주문/결제/리뷰/관리자 페이지 등 향후 기능 확장이 용이한 구조

## 핵심 화면 (MVP)

| 화면 | 목적 | 주요 표시 정보 |
|---|---|---|
| 상품 목록 | 다양한 커피 상품 탐색 | 대표 이미지, 상품명, 가격/할인가/할인율, 평점, 리뷰 개수, 배송 정보, BEST/NEW, 품절 여부 |
| 상품 상세 | 상품 상세 정보 확인 | 상품 이미지, 설명, 원산지, 무게, 유기농 인증 소개, 리뷰, 장바구니 담기 버튼 |
| 장바구니 | 구매 예정 상품 관리 | 상품 목록, 수량 변경, 상품 금액, 배송비, 총 결제금액, 주문하기 버튼 |
